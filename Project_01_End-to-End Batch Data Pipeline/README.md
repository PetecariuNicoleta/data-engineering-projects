# End-to-End Batch Data Pipeline

## Overview
This project implements an end-to-end batch data pipeline using a lakehouse architecture.

## Architecture
- Raw data ingested from CSV files
- Bronze, Silver, and Gold layers implemented using Delta Lake
- Data modeling using a star schema

## Technologies
- Python
- PySpark
- SQL
- Databricks
- Delta Lake

## Key Concepts
- ETL / ELT
- Batch processing
- Incremental-ready design
- Dimensional modeling
